## Imagen

### Usando Markdown

La [sintaxis de markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes usando el siguiente código

```md
![title](/path/image.jpg)
```

Ten en cuenta que el archivo de imagen debe ser menor a 5MB. De lo contrario, recomendamos alojarla en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Luego puedes usar esa URL e incrustarla.

### Usando elementos incrustados

Para obtener más personalización con las imágenes, también puedes usar [embeds](/writing-content/embed) para añadir imágenes

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Carga otra página HTML dentro del documento. Se usa con mayor frecuencia para incrustar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
